# LevitateOS Website

Documentation portal for LevitateOS built with TanStack Start and React 19.

## Quick Start

```bash
bun install && bun run dev  # http://localhost:3000
```

## Features

- Template-driven documentation system
- Shiki syntax highlighting
- Responsive design with Tailwind CSS 4

## Development

```bash
bun run dev         # Start dev server
bun run build       # Production build
bun run typecheck   # TypeScript validation
bun run check       # Full validation (typecheck + lint + format)
```

## Tech Stack

React 19, TanStack Start, Tailwind CSS 4, shadcn/ui, Shiki
